## Visual Feedback Loop（標準手順）

Claudeがマイルストーンに到達したとき、または設計判断が必要なとき:

```
1. BRIDGE.md を更新（フェーズ状況・既知の問題・Geminiへの質問を記載）
2. Chrome MCP で gemini.google.com を開く
3. 該当チャット（プロジェクト専用スレッド）を特定
4. Quill エディタに質問を execCommand('insertText') で注入
5. 送信ボタンをクリック（aria-label="プロンプトを送信"）
6. get_page_text でGeminiの回答を取得
7. BRIDGE.md の [Answer] セクションに転記
8. 回答に基づいて実装を継続
```

### 既知の技術的注意点

- Gemini の入力欄は `div.ql-editor`（Quillエディタ）→ `innerHTML` は Trusted Types でブロックされる
- 正しい注入方法: `document.execCommand('insertText', false, msg)`
- 送信ボタン: `aria-label="プロンプトを送信"` のボタン要素
